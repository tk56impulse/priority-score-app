STRATEGIC LAYER (ストラテジック・レイヤー)
「何からやるか」に迷わない、生存戦略型タスク管理ツール
独自のスコアリングアルゴリズム（熱量 × カテゴリ × 戦略レイヤー）を用いて、膨大なタスクの中から「今、自分の価値観に沿って着手すべきもの」を可視化するWebアプリケーションです。

1. アプリ概要
   目的: タスクの優先順位を「主観（やりたい）」と「客観（やらなきゃ）」の両面から計算し、納得感のある行動順位を提示する。

課題解決: タスクが多すぎて何から手をつければいいか分からない、または「緊急だが重要でないこと」に時間を奪われ、自分の「本音」が後回しになる問題を解決します。

2. こだわりの機能（開発の工夫）
   🚀 多次元優先順位エンジン
   単なるToDoリストではなく、ユーザーの「生き方」を選択に反映させる独自のスコアリングシステムを実装しています。

① 三層の戦略レイヤー × ライフカテゴリ
タスクを人生における役割（レイヤー）で分類し、スコアに動的な倍率補正をかけます。

絶対 (Deadline): 生存や社会生活に不可欠なタスク。

投資 (Investment): 未来の自分のための種まき。

本音 (Desire): 自分の純粋な欲求・ご褒美。
さらに「仕事」「自己研鑽」「趣味」のカテゴリを掛け合わせることで、多角的な評価を行います。

② 「味付け（Appraisal Mode）」による動的解析
解析時に「甘口・普通・激辛」を選択することで、同じリストでも結果が劇的に変化します。

🍬 甘口（褒めちぎり）: 「本音」レイヤーに1.6倍のブーストをかけ、自分の欲求を最優先したポジティブな順位を生成。

⚖️ 普通（的確）: 標準的なロジックでバランス良く解析。

🌶️ 激辛（魂の解析）: 「絶対」レイヤーを2.0倍で評価し、本音のスコアを0.5倍に圧縮。「現実」を突きつけるストイックな順位を生成。

③ 心理的UXの「引き算」
直感的な「熱量」入力: 入力画面ではあえて「スコア（数字）」を非表示にし、視覚的な「熱量（💤 ⚡ 🔥アイコン）」のみを提示。ユーザーが数字をハックしようとする心理を排除し、自分の直感に従った入力を促します。

判断コストの削減: 以前の煩雑な設定を廃止。レイヤーを選択し、1本の「熱量スライダー」を動かすだけで、裏側で複雑な計算が行われるシンプルかつ深いUXを追求しました。

3. 機能詳細
   タスク管理: 直感的な追加/削除。未入力タスクのバリデーション機能付き。

期日ボーナス: 締切3日前からスコアを加算し、直近のタスクを確実に上位へ浮上。

動的ランキング: スコア順にリアルタイムソートし、1位のタスクを強調。

モード別フィードバック: 解析モードに合わせたAI的メッセージを表示。

データ永続化: LocalStorageを活用し、ブラウザを閉じてもデータを保持。

4. 技術スタック
   Frontend: Next.js 15 (App Router), TypeScript

State Management: React useState, useMemo, useEffect

Logic: TypeScriptによる厳格な型安全性の確保（Record, keyof, Union型 の活用）

Storage: LocalStorage API

Design: CSS-in-JS (Inline Styles) による軽量なスタイリング

5. 開発環境のセットアップ
